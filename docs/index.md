# OrbisAlert（オービスアラート）

**オービス接近を音声とアラートで通知する Android アプリです。**

---

## 主な機能

- **現在地に応じて自動でオービスを検出**
- **音声・アラート・バイブレーションで通知**
- **Google マップ上にオービスを表示**
- **高速道路・一般道に応じた警告音を切り替え**
- **バックグラウンドでも検知が可能（運転中にアプリを閉じてもOK）**
- **バッテリー消費を抑える設計（状況に応じてGPSの更新間隔を自動調整）**
- **無料で利用可能（AdMobによる広告収益モデル）**
